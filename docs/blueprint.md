# **App Name**: TaskFlow

## Core Features:

- Email Task Extraction: Email Parsing Tool: Automatically extract and separate multiple tasks from a single email received at the PMO email ID. This tool utilizes reasoning to understand the task requirements, vertices (CMIS, LOF, TRI, TRG), start date, end date and review stages from the email content.
- Automated Ticketing: Task Ticketing: Generate individual task tickets with details extracted from the email and display the information on an interface.
- Task Assignment: Task Allocation UI: A PMO interface to allocate tasks to individual team members or groups.
- Email Notifications: Notifications: Send email notifications to team members upon task allocation.
- Progress Updates: Task Progress UI: Allow team members to update the completion percentage of tasks daily.
- Real-time Dashboard: Automatically update a dashboard to visualize task status, completion rates, and potential delays. Provide views for each vertices (CMIS, LOF, TRI, TRG). Daily task updates to each vertices as per progress automatically
- Cost Estimation: Automatically generate the cost of each task and total cost of each vertices according to PMO inputs (days, Hours) in excel format.

## Style Guidelines:

- Primary color: Teal (#008080) to represent productivity and reliability. We're moving beyond cliches and taking Teal as the heart of the experience.
- Background color: Light gray (#E0E0E0) to provide a neutral and clean backdrop for content.
- Accent color: Green (#00FF00) for positive actions like task completion and approvals, to give a high-tech feeling that pops.
- Body and headline font: 'Inter' sans-serif font to maintain clarity and modern readability across all text elements.
- Use clear and functional icons for task status, actions, and categories.
- Design a clear, intuitive layout for the task board and dashboard. Use clear grouping to separate tasks and summaries for the different verticles. Prioritize key information for quick understanding.
- Implement subtle transitions and animations to enhance user experience, such as task status changes and dashboard updates.
